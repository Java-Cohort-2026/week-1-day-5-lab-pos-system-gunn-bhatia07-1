# W1 Friday Lab: The Cold Brew POS System

### Your Mission
Open `src/Main.java`. You must build a `while` loop that does the following:
1. **Display a Menu** exactly like this:
   `[1] Cold Brew ($4.50)`
   `[2] Iced Latte ($5.00)`
   `[0] Finish Order`
2. **Read the Input:** Wait for the user to type 1, 2, or 0.
3. **Calculate:** Add the appropriate price to the `totalCost` variable based on their choice.
4. **Repeat:** Keep showing the menu and asking for inputs until they type `0`.
5. **Finish:** When they type `0`, exit the loop and print the final total.
